🚀 Cómo Empezar

Paso 1: Adquiere tu Herramienta

Asegúrate de tener Node.js instalado. Luego, clona el repositorio o descarga el código fuente y navega al directorio del proyecto:

$ git clone https://github.com/your-user/immortal-ia-dev-cli.git
$ cd immortal-ia-dev-cli
$ npm install

Esto instalará todas las dependencias necesarias para que el CLI funcione.

Paso 2: Conecta la Inteligencia

Para desatar el poder de la IA, necesitarás una API Key de Google Gemini (o OpenAI GPT). Crea un archivo .env en la raíz del proyecto y añade tu clave:

GOOGLE_API_KEY="TU_API_KEY_DE_GEMINI_AQUI"

Registra tu clave de API en Google AI Studio.

Paso 3: ¡Ejecuta el Inmortal!

Con todo configurado, lanza el CLI desde tu terminal. El menú interactivo te guiará por todas las opciones:

$ npm start

¡Felicidades, estás listo para crear proyectos a la velocidad de la luz!
