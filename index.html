🚀 Fases de Activación

Fase 1: Recopilación de Componentes Base

Accede a los archivos operativos del Immortal IA Dev CLI. Asegúrate de tener Node.js instalado (Versión estable recomendada: v18+).

$ git clone https://github.com/your-user/immortal-ia-dev-cli.git
$ cd immortal-ia-dev-cli
$ npm install --force

El comando --force asegura la resolución de dependencias, crucial para el correcto despliegue.

Fase 2: Conexión Neural de IA

Para la integración con el Motor Cognitivo IA, se requiere un Identificador de Acceso. Crea un archivo .env en la raíz del directorio de operación:

GOOGLE_API_KEY="TU_API_KEY_DE_GEMINI_AQUI"

Obtén tu Identificador de Acceso en Google AI Studio. Se recomienda un monitoreo de uso.

Fase 3: Iniciación del Protocolo Principal

Con los componentes base y la conexión IA establecida, lanza el CLI para iniciar la interfaz de operaciones:

$ npm start

La interfaz te proporcionará un menú táctico para seleccionar tu primera operación de desarrollo. Prepárate para una eficiencia sin precedentes.
